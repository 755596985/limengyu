# 小鱼 ❤ 小鱼 - 情侣空间

一个温馨的情侣纪念网站，记录属于两个人的点点滴滴。

## 功能介绍

- **纪念日计时**：自动计算在一起的天数，实时展示
- **说说**：发布动态，支持图片和视频，带评论互动
- **相册**：记录美好瞬间
- **足迹**：标记一起去过的地方
- **清单**：共同完成的小目标
- **用户系统**：注册、登录、个人中心

### 后台管理

- 站点配置（名称、头像、称呼等）
- 文件管理（浏览、上传、编辑、删除）
- 敏感词过滤（留言和说说自动过滤）
- 访客记录（IP、时间、来源页面）
- 版本介绍（更新日志管理）

## 技术栈

- 后端：PHP + MySQL
- 前端：原生 HTML/CSS/JS，移动端响应式适配
- 安全：CSRF 防护、XSS 过滤、SQL 预处理语句、路径穿越防护

## 安装教程

1. 将代码部署到 PHP 7.4+ 环境中
2. 导入 `install/schema.sql` 到 MySQL 数据库
3. 配置 `include/config.db.php` 中的数据库连接信息
4. 确保 `data/` 目录有写入权限
5. 访问首页完成初始化

## 目录结构

```
├── index.php           # 首页
├── login.php           # 登录
├── register.php        # 注册
├── user.php            # 用户中心
├── migrate.php         # 数据库迁移
├── admin/
│   ├── index.php       # 后台登录
│   └── manage.php      # 后台管理面板
├── include/
│   ├── bootstrap.php   # 核心函数库
│   └── config.db.php   # 数据库配置
├── data/               # 数据文件目录
├── install/
│   └── schema.sql      # 数据库初始化脚本
└── uploads/            # 用户上传文件
```
